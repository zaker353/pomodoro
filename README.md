# 🍅 番茄專注鐘

番茄鐘 + 白噪音的離線專注工具(個人自用)。

- 純離線 PWA,資料只存在裝置本機(localStorage),支援「加入主畫面」
- 白噪音(雨聲/海浪/咖啡廳/火堆/遠雷)使用真實環境錄音,離線快取;音檔讀不到時自動退回 Web Audio 程式合成
